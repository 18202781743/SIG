# DDE 桌面环境小组

## 小组简介

DDE 桌面环境小组是对 DDE 桌面环境进行开发与维护的小组。

## 活动范围与目标

DDE 桌面环境小组活动内容以 DDE 本身的开发、缺陷修复与维护为主，研发活动位于 GitHub 进行开展，研发相关的内容讨论位于 Matrix。

此小组旨在确保 DDE v23 起的研发活动的稳健开展，确保 DDE 本身质量，并吸引贡献者一同参与 DDE 的完善之中。
